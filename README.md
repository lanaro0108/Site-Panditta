## 📌 Sobre o projeto

A **Panditta Store** é uma landing page de e-commerce desenvolvida com foco em **design sofisticado**, **experiência do usuário (UX)** e **interface moderna (UI)**.

O projeto simula uma vitrine digital, apresentando produtos de forma elegante e intuitiva, com navegação fluida e responsiva.

> 💡 Este projeto foi criado com o objetivo de demonstrar habilidades em desenvolvimento front-end e construção de interfaces profissionais.

---

## 🚀 Principais funcionalidades

- 📱 **Design responsivo completo** (mobile-first)  
- 🎬 **Animações suaves on-scroll** com *Intersection Observer*  
- 🧭 **Menu de navegação interativo** (desktop + mobile)  
- 🛍️ **Cards de produtos dinâmicos**  
- 🎯 **Destaque automático da seção ativa**  
- 🔗 **Integração com redes sociais e marketplace**  
- ⚡ **Scroll suave entre seções**

---

## 🧠 Decisões técnicas

O projeto foi estruturado priorizando:

- Separação de responsabilidades (HTML + CSS + JS)  
- Uso de variáveis CSS para padronização visual  
- Performance e leveza (sem frameworks)  
- Escalabilidade para futuras integrações com backend
